# Generated Data Files

This directory contains JSON files that are **generated automatically by scripts**.

Do not manually edit these files unless you know what you are doing.

They are regenerated by scripts in the `scripts/` directory.

## Files

gallery.json
Generated from a Google Drive photo folder.

instagram.json
Generated by the Instagram scraping script.

jobs.json
Generated from a Google Sheet used by the NYRG jobs form.

## Updating data

Use the scripts in:

scripts/

Examples:

Update gallery:

./scripts/update_gallery_json.sh <GOOGLE_DRIVE_FOLDER_ID>

Update jobs:

./scripts/update_jobs_daily.sh

Update Instagram:

./scripts/daily_instagram_update.sh
